# React Starter

Sample app using React, Redux, and Webpack 2.

## Contains

- [x] [Webpack 2.2](https://webpack.github.io)
- [x] [React](https://facebook.github.io/react/)
- [x] [Redux](http://redux.js.org/)
- [x] [Babel](https://babeljs.io/)

## Setup

```
$ yarn install -E
```

## Running in dev mode

```
$ yarn start
```

## Running in production mode

```
$ yarn run prod
```

## Build (production)

```
$ yarn run build
```
